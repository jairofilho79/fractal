This must be of the same size of the another polygons. The dots must be Numbers!



If you want to use Iteration, use i


Ex: [[(p.x1)/2,(p.y1)/3],[(p.x2)*2,(p.y2)+10],[(p.x3)*(40/100),(p.y3)-((p.y1)+(p.y2))]]
The first `[]` means the polygon, each `[]` inside polygon means dot. The number of dots determine the type of polygon. `p` is the object to access the preview value.
